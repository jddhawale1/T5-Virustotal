# T5-Virustotal
Malware Types & Behavior Analysis
Overview

This project analyzes different types of malware and their behavior using VirusTotal to understand how malicious software works and how it can be detected.

Tools

VirusTotal

Key Learnings

Classification of malware (Virus, Worm, Trojan, Ransomware, Spyware, Adware)

Malware behavior and lifecycle

Detection and prevention methods

Outcome

Improved understanding of malware threats and basic cybersecurity defense techniques.
